src.medichat.eval.calculate_answer_similarity(answer: str, source_answers: List[str]) → float[source]

Calculate semantic similarity between chatbot answer and source answers.

Uses sentence transformers to embed texts and cosine similarity for comparison.

Parameters:

  • answer (str) – The chatbot’s generated answer.

  • source_answers (List[str]) – List of reference answers from sources.

Returns:

The highest similarity score (0-1) between the answer and any source answer.

Return type:

float

src.medichat.eval.main()[source]

Main execution function for the evaluation system.

This function: 1. Runs the evaluation 2. Displays results in the terminal 3. Saves detailed results to JSON and text files

The results are saved in two formats: - A JSON file containing structured results and metadata - A text file with detailed comparisons

src.medichat.eval.run_evaluation() → DataFrame[source]

Run the complete evaluation process for the chatbot.

This function: 1. Creates necessary directories 2. Loads test data 3. Tests the chatbot on each question 4. Saves detailed comparisons 5. Calculates similarity scores 6. Measures response times

Returns:

Results containing questions, answers, similarity scores, and response times.

Return type:

pd.DataFrame
